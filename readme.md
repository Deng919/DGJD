这是电工基地寒假的小组课题，题目是车牌识别并控制舵机抬杆

技术路线：
1.硬件：USB摄像头，arduino R3板子，普通舵机，蜂鸣器，OLED显示屏，面包板，杜邦线若干
2.实现方式：
<1>通过USB摄像头抓取含有车牌的视频流，传到PC上
<2>调用openCV通过锁定颜色阈值过滤非目标区域，进行形态学闭运算消除边缘检测后的细小间隙，让车牌轮廓更完整，进行轮廓筛选，从面积阈值，长宽比矩形锁定有效ROI，二值化处理转为黑白图像增大对比度
<3>使用HyperLPR开源车牌识别库返回车牌号码和置信度，仅保留置信度大于0.5的结果
<4>进行合法性校验,匹配车牌的正确格式，白名单核对
<5>增加检测计数器，只有连续检测才触发识别，同时5帧识别一次减低算力消耗，同一车牌仅发送一次指令，定期清空缓存
<6>摄像头资源释放（cap.release ()）串口资源关闭（arduino_serial.close ()）窗口销毁（cv2.destroyAllWindows ()）异常捕获：串口 / 摄像头初始化失败时给出明确提示，程序不崩溃

用到的库：OpenCV，numpy，serial，imutils，hyperlpr，sys，time，rs
所有用到的库版本都放在ku文件中，python版本为3.9
